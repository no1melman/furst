# Furst

Functional first memory safe language based on LLVM.

## Getting Started

Run

```pwsh
./build.ps1
```

This will pull down a version of LLVM into a `deps` directory. It will also create a build directory and run `cmake` into it  

### Tooling

  - CMake 
  - CCache
  - Ninja
  - Clang & Clangd 
  - C++ 20
